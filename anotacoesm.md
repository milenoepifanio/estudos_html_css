## CSS
### Maneiras de adicionar CSS
- Inline: quando o CSS é adicionado a um atributo
- Internal: Quando é adicionado a uma tag head
- External: Quando é adicionado atravrez de um arquivo externo, e importado

- Propridades (cor, tamanho, dimensão, espaçamento) e valores.

- Ordem do CSS: inline > internal = external > padrão do navegador

- Classes e ids
    - Ligados ao HTML, mas não obrigatoriamente ao CSS
    - ids são únicos por páginas
    - Podemos escrever qualquer coisa como valor, será o nome do id
